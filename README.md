# Business Name Generator 🎯

A simple web app that generates random business names using JavaScript.

## 🚀 Features
- Random name generation
- Simple and clean UI
- Lightweight project
- Beginner-friendly

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 💡 Example Output
- Ramji ke Chole Bhature
- Arunji ki Chai Paani Free
- Shyamji ke Sev Pyaz Pohe

## 📌 How to Use
1. Click on "Generate Name"
2. Get a random business name instantly
